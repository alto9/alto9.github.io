What next?¶

You have gone through the Quickstart installation: at this point you should have a running OpenCart platform. If you don’t, please follow the instructions from the Troubleshooting section.

Logging-in as administrator¶

Out of the box, Peddler only creates the admin user for you. You may want to create another user for yourself, or others using the OpenCart admin UI.

Making OpenCart look better¶

Peddler theming is under development.

Deploying to Kubernetes¶

Yes, Peddler comes with Kubernetes deployment support out of the box.
